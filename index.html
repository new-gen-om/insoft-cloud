Cloud Mesh

CloudMesh with Multi-Cloud

다양한 Cloud와 연동 개발하여 Insoft 만의 CMP 표준을 만들어 가고 있습니다. 퍼블릭, 프라이빗, 하이브리드 클라우드를 단일 플랫폼에서 통합 관리하세요.

  • ✓ IaaS - VMware, Nutanix, OpenStack, PowerVC 통합 관리
  • ✓ PaaS - Kubernetes Multi Cluster (EKS, AKS, GKE, Tanzu) 운영
  • ✓ Public - AWS, Azure, GCP 하이브리드 클라우드 관리
  • ✓ Automation - vRealize, Terraform IaC 자동화

EXODUS

OpenStack Private Cloud Platform

EXODUS는 OpenStack 기반의 엔터프라이즈 프라이빗 클라우드 플랫폼입니다. Appliance 형태로도 제공되어 빠르고 간편한 도입이 가능합니다. 검증된 아키텍처로 안정적인 클라우드 인프라를 구축하세요.

  • ✓ OpenStack 기반 검증된 플랫폼
  • ✓ Appliance 형태 제공
  • ✓ 엔터프라이즈 수준 보안
  • ✓ 유연한 확장성

ACTs

Automated CNP Tool EXODUS

ACTs는 EXODUS 플랫폼 위에 구축된 Cloud Native Platform 솔루션입니다. 컨테이너 기반 애플리케이션의 자동화된 배포와 관리를 지원하며, Kubernetes와 OpenStack을 통합하여 현대적인 클라우드 네이티브 환경을 제공합니다.

  • ✓ iCNP - Cloud Native Platform 통합

  • ✓ EXODUS Controller & Compute 아키텍처
